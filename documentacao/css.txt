CSS: folha de estilo em cascata
style: estilo do elemento
style="font-size: ": font-size por padrao tem 16px
text-align: para realizar o alinhamento do texto na pagina, "center" = centro
para passar mais de um stylo da pagina no style usar ; ponto e virgula
# : referencia ao id de estilo de CSS
. : referencia a classe de estilo CSS
width: largura do  elemento na pagina
height: altura do elemento na pagina
border: "espessura" "textura" "cor": borda do elemento "border: 10px solid #000000;
padding: "espessura" : espaçamento interno. padding: 20px; padding: 20%; padding-top: 20px; espassamento no topo;
marging: espaçamento externo. marging-left; margem da esquerda. marging-rigth. margem direta.
display: block; ocupa 100% da largura da pagina. todos elementos tem block
display: inline; nao deixa alterar espaçamento interno nem espaçamento externo.
display: inline-block: bloqueia uma largura, sou eu qem defino o tamanho
vertical-align: top; caracteristica do display:inline-block; alinhado por baixo, "top" alinha pelo topo.
text-transform: uppercase; transforma letras em maiusculas;
color: black; Cor do elemento;
font-weight: espessura bolder negrito
